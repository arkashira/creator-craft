# Project Manager
A simple project management dashboard.

## Usage
1. Create a new project: `project_manager.add_project('Test Project', 'https://example.com')`
2. Rename a project: `project_manager.rename_project('Test Project', 'New Name')`
3. Duplicate a project: `project_manager.duplicate_project('Test Project')`
4. Delete a project: `project_manager.delete_project('Test Project')`
5. Get all projects: `project_manager.get_projects()`
6. Update project status: `project_manager.update_status('Test Project', 'Live')`
